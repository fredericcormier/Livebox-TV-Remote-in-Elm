# Livebox TV Remote/ Télécommande Livebox TV

##
### Written in Elm

Create a js directory then compile with
`elm-make ./elm/Livebox.elm  --output=./js/Livebox.js`

Enter your TV Decoder IP Address

Enjoy!!!
